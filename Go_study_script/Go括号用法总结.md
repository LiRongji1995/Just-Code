
# Go语言括号使用总结

## ✅ 括号类型概览

| 括号 | 类型 | 常见用途 | 示例 |
|------|------|----------|------|
| `()` | 圆括号 | 函数调用、函数参数、类型转换 | `add(1, 2)` |
| `{}` | 花括号 | 代码块、复合字面量（结构体/map） | `if x > 0 { ... }`, `Person{Name: "Tom"}` |
| `[]` | 中括号 | 切片、数组、索引、map 访问 | `arr[0]`, `m["key"]` |

---

## 🎯 详细解析

### 1. 函数声明与调用

- **函数声明**：
  ```go
  func add(a int, b int) int {
      return a + b
  }
  ```

- **函数调用**：
  ```go
  result := add(1, 2)
  ```

---

### 2. 控制结构（**不加圆括号！**）

```go
// 正确 ✅
if x > 0 {
    fmt.Println("positive")
}

// 错误 ❌
if (x > 0) {
    fmt.Println("wrong")
}
```

> Go 中 `if`、`for`、`switch` 的条件表达式不使用 `()`。

---

### 3. 复合字面量

```go
arr := []int{1, 2, 3}              // 切片
m := map[string]int{"a": 1}       // map
p := Person{"Alice", 30}          // 结构体
```

---

### 4. 索引与切片操作

```go
arr := []int{1, 2, 3}
val := arr[0]         // 索引访问
sub := arr[1:3]       // 切片操作
```

---

## 🧠 快速记忆口诀

```
圆括号传参数，控制结构别乱加；
中括号看集合，访问索引和切片；
花括号最常见，函数代码和结构；
```
